Status of Rules applied to this node

The table below show the current status for each Rule applied to this node, as reported by it's agent when it last ran.
Each line represents one Rule, and may be in the following statuses:

  • Success: The system is already in the desired state. No changes were required. Conformity is attained.
  • Repaired: The system was not in the desired state. Rudder applied some changes and repaired incorrect settings. The system is now in the desired state. Conformity is attained.
  • Error: The system was not in the desired state. Rudder tried to apply some changes, but failed. The system could not be repaired.
  • Applying: When a Rule is first applied, Rudder waits for 10 minutes for a report. During this period, the Rule's status is Applying.
  • No answer: No reports were received from this node for this Rule for over 10 minutes. Conformity is unknown - the node may be non responsive, or temporarily isolated from the network.
