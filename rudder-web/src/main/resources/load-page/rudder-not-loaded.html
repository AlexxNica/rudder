Page not found

This may be caused by:
  • A non-existing URL.
  • An essential component (LDAP or PostgreSQL) that is not accessible.
  • The web application being unable to load.
  • A non initialised Rudder installation. Please run /opt/rudder/bin/rudder-init to set it up.
  • Not space left on disk, Rudder may have been stopped to prevent data corruption. Please check that you have available space.
More information can be found in the Rudder webapp log file in /var/log/rudder/webapp/ on the Rudder server.
